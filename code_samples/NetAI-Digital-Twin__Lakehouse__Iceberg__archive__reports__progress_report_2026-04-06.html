Design decisions

  • Separate Iceberg namespace (nvidia_cosmos) — droppable without affecting base lakehouse
  • Read-only dependency on Gold — never writes to nvidia_bronze/silver/gold
  • Dual backend — cloud API avoids GPU requirement during development; NIM container for production with dedicated hardware
  • Config-driven variations — new conditions added by editing a dict, no code changes
